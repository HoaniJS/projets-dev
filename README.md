# Site concepts

## Contexte
Ce projet est le résultat d'un exercice fait à EFREI, visant à s'entraîner à React.

L'objectif était de créer un site en React avec Vite hébergeant plusieurs "concepts" :

- Un **compteur** qu'il est possible d'incrémenter, de décrémenter, et de réinitialiser à partir de boutons. On peut également changer la couleur du texte du compteur depuis n'importe quelle page du site.
- Deux **tableaux** dont on peut changer le contenu en cliquant sur leurs en-tête.

## Installation
1. Cloner la branche
2. Accéder au répertoire depuis la console
3. Entrer la commande `npm install`
    - Si Node n'est pas installé, l'installer d'abord
4. Entrer la commande `npm run dev` et accéder à l'adresse affichée par la console
