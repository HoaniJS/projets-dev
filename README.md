# Todo List

## Contexte
Ce projet est le résultat d'un exercice fait à EFREI, visant à s'entraîner à React et à la manipulation d'API.

L'objectif était de créer une application React avec Vite affichant une blague sur Chuck Norris (_Chuck Norris fact_) aléatoire, et permettant de l'enregistrer/la supprimer d'une liste de favoris.

L'énoncé complet de l'exercice est disponible [ici](https://github.com/HoaniJS/projets-dev/blob/efrei.chuck-norris/instructions.md).

## Installation
1. Cloner la branche
2. Accéder au répertoire depuis la console
3. Entrer la commande `npm install`
    - Si Node n'est pas installé, l'installer d'abord
4. Entrer la commande `npm run dev` et accéder à l'adresse affichée par la console

## Note
Ce projet a été réalisé entre septembre et octobre 2025, avant la mort de l'acteur Chuck Norris le 19 mars 2026.
