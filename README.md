# Mini application en JavaScript avec JSONPlaceholder

## Contexte
Ce projet est le résultat d'un exercice fait à EFREI, visant à s'entraîner à la manipulation d'API REST, à la manipulation du DOM, à la gestion des états chargement/erreur/vide, et à la navigation simple entre plusieurs vues sans framework.
L'objectif était d'afficher une liste d'utilisateur dans une colonne à gauche, les posts d'un utilisateur cliqué à droite, et les détails et commentaires d'un post cliqué à droite. 

## Installation
1. Cloner la branche
2. Ouvrir le fichier `index.php` dans un navigateur
