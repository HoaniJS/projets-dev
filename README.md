# Sélecteur de photos

## Contexte
Ce projet est le résultat d'un exercice fait à EFREI, visant à s'entraîner au JavaScript.

L'objectif était d'afficher une liste de photos sélectionnable d'un clic de souris, avec un message indiquant le nombre de photos sélectionnées à chaque instant.

L'énoncé complet de l'exercice est disponible [ici](https://github.com/HoaniJS/projets-dev/blob/efrei.selecteur-photos/INSTRUCTIONS.md)

## Installation
1. Cloner la branche
2. Ouvrir le fichier `index.php` dans un navigateur
