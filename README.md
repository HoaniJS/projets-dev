# Gestion de tâches

## Contexte
Ce projet est le résultat d'un test fait à EFREI, concluant la matière **Renforcement technique**. Ce test avait pour but d'évaluer notre niveau en JavaScript/TypeScript.

L'objectif était de concevoir une application web orientée objet permettant de créer, afficher, modifier et supprimer des tâches.

Pour cause de manque de temps (devoir donné à 17h et à rendre le lendemain avant 12h), je n'ai pu faire que la partie front-end en HTML "en dur", sans JavaScript. L'application est donc dépourvue de toute intéraction.

Certains éléments de la page ne sont pas affichés car ils devraient normalement l'être par du JavaScript selon certaines conditions. Pour les afficher, vous pouvez modifier le fichier `index.html` ou inspecter la page dans le navigateur et supprimer le mot-clef `d-none` des classes où il est présent.

## Installation
1. Cloner la branche
2. Ouvrir le fichier `index.html` dans un navigateur
